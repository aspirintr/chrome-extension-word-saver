Attach tooltips to text selection

Popper.js can be used to position anything, even tooltips that need to be positioned near a selected text!
Try to select part of this article to see a tooltip appear near it.

You can also try to select a text, then select something else, and see the tooltip automatically update to being properly positioned.
If you scroll this text while the tooltip is visible, you'll notice how it will automatically flip to stay between the viewport.
